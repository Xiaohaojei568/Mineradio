## 说明

- Windows 用户优先下载 `Setup.exe`。
- Mac 用户优先下载对应芯片版本的 `dmg`。
- 不确定 Mac 芯片类型时，打开左上角苹果菜单，选择“关于本机”查看芯片信息。
- 如果推荐下载打不开，就换备用下载。
- 已安装 v1.2.3 或后续版本的用户，之后可以优先使用软件右上角更新按钮。
