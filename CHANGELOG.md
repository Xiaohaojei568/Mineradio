# Changelog

## v1.1.1

- 新增汽水音乐端：支持读取本机汽水音乐登录会话，接入汽水搜索、播放链接、歌词、歌单、首页推荐和“我喜欢的音乐”等核心链路，功能体验对齐网易云和 QQ 音乐。
- 多端播放逻辑重整：单端登录时优先使用当前端播放，多端登录时自动按可用音源兜底；汽水、QQ、网易云的播放失败提示会标明真实来源，不再误报成其它平台的音质降级。
- 会员与音质状态改为实时真实数据：有 VIP/SVIP 才显示对应标识，没有会员不显示；播放时默认选择当前账号可用的最高音质，不可用规格灰化且不可切换。
- 会员歌曲试听逻辑统一：没有会员但平台返回试听片段时不再强行拦截，播放器会切到试听播放并明确提示用户当前是试听片段。
- 修复播放失败后控制条歌曲和实际播放音频不一致的问题，避免点击播放按钮时继续播放上一首可播放歌曲。
- 优化汽水歌词、拖动视觉区域后的歌词同步，以及汽水端切换/回首页时右侧布局短暂错乱的问题。
- 优化左侧歌单抽屉，多端登录后不再展示多余登录提示，歌单和详情数据按当前端真实状态刷新。
- 优化应用进入和使用过程中的卡顿，减少后台/前台切换、视觉恢复、面板刷新和账号同步造成的重复渲染。
